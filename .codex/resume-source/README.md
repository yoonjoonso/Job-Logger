# Codex Resume Source

This folder is reserved for optional Codex-side resume source material that a user may choose to keep locally.

## Intended Use

Use `.codex/resume-source/` for private, user-specific source inputs that help Codex understand how to tailor resumes, such as:

- parsed resume notes
- structured candidate fact sheets
- local working copies derived from personal documents

These files are not required for the public setup flow. A new user can start from the repo setup script and populate their own local inputs later.

## Git Boundary

Do not commit personal resume source material here.

If this folder is used locally, keep only public-safe documentation in the repo and treat actual contents as user-owned private state.
