# Shared Handoff

This folder is only for short cross-agent handoff notes.

Files:

- `for-claude.md` - notes Codex leaves for Claude
- `for-codex.md` - notes Claude leaves for Codex

Guidelines:

- Add entries after any meaningful repo change that may affect the other agent.
- Keep entries short and actionable.
- Use newest-first ordering.
- Mark items `new` when written.
- The receiving agent can change `new` to `read` or clear resolved items.
- Do not use this folder for general design docs, audits, or private agent notes.
- The live handoff files are local-only and are gitignored for public use.

Required fields for each entry:

- `Status`
- `Files`
- `Change`
- `Why`
- `Follow-up` when applicable
